|  符号   | 字符含义  |
|  :----:  | :----:  |
|  \n  | 换行 (0x0a) |
|  \r  | 回车 (0x0d) |
|  \f   | 换页符(0x0c)  |
|  \b   | 退格 (0x08)  |
|  \0   | 空字符 (0x20)  |
|  \s   | 字符串  |
|  \t   | 制表符  |
|  \"   | 双引号  |
|  \'   | 单引号  |
|  \\   | 反斜杠  |
|  \ddd   | 八进制字符 (ddd)  |
|  \uxxxx   | 16进制Unicode字符 (xxxx)  |
