# 第一个Dart程序
新建文件hello.dart，如下：
```
// 定义一个函数
printString(String msg) {
  print(msg);
}

main() {
  //声明和初始化变量
  var str = 'Hello World! Welcome to autoref.cn!';
  //函数调用
  printString(str);
}
```
其中main()函数是必须的顶层函数，是程序执行的入口。

执行命令：
```
$ dart hello.dart
```

执行输出结果为：
```
Hello World! Welcome to autoref.cn!
```
