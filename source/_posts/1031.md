1、安装nodejs和npm

2、安装appium
``` bash
  $ npm install -g appium
```

3、安装Appium Python Client
``` bash
  $ pip install Appium-Python-Client
```

4、验证
``` bash
  $ appium-doctor
```

5、启动Appium
``` bash
  $ appium
```

6、开启测试
``` bash
  $ py.test android_simple.py
```
